Forces the exit of all processes in MPI_COMM_WORLD.

Author
Michael Duda
Date
26 March 2013

This routine writes the specified message to standard error and to a per-process log file named log.????.abort. However, if the optional argument deferredAbort is set to .true., messages will be written to standard error and to the log.????.abort files, but MPI tasks will not be termintated. This allows code to write several lines of messages before exiting.

Parameters
[in]mesgInput: Abort message
[in]deferredabortInput: Defer call to abort until later
