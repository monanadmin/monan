subroutine mpas_field_routines::mpas_shift_time_levs::mpas_shift_time_levs_0dchar ( type (field0dchar), dimension(:), pointer  fldarr)

MPAS 0D character time-level shift routine.

Author
Michael Duda
Date
04/14/14

Shifts the contents of the array of fields provided by the input argument. After returning, the storage for fldarr(n) will point to what was the storage for fldarr(n+1) in a period fashion, so that, for N time levels, the storage for fldarr(N) will point to what was the storage for fldarr(1).
