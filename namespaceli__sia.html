Computes velocity using Shallow Ice Appoximation.

Author
Matt Hoffman
Date
21 May 2012

This routine computes the normal velocity on edges for each layer using the Shallow Ice Approximation. It calculates ice thickness on on an edge using the average of the two neighboring cells (2nd order).

Parameters
[in]meshpoolInput: mesh information
[in]geometrypoolInput: geometry information
[in]thermalpoolInput: thermal information
