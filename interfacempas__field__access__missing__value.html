Accesses the 'missingValue' member for a field given the field name.

Author
Doug Jacobsen, Michael Duda
Date
28 March 2016

This routine returns the value of the 'missingValue' member from the field type for the specified field. The named field must exist in the specified pool; if it does not, an error message will be printed.
