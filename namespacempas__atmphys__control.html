Checks physics input fields and options for compatibility.

Author
Miles Curry and Michael Duda
Date
25 October 2018

This routine checks the input fields and run-time options provided by the user for compatibility. For example, two run-time options may be mutually exclusive, or an option may require that a certain input field is provided. The checks performed by this routine are only for physics related fields and options.

A value of 0 is returned if there are no incompatibilities among the provided input fields and run-time options, and a non-zero value otherwise.
