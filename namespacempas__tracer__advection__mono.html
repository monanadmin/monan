MPAS monotonic tracer advection tendency with FCT.

Date

This routine computes the monotonic tracer advection tendencity using a FCT. Both horizontal and vertical.

Parameters
[in]tracersInput: current tracer values
[in]adv_coefsInput: Advection coefficients for 2nd order advection
[in]adv_coefs_3rdInput: Advection coefficients for blending in 3rd or 4th order advection
[in]nadvcellsforedgeInput: Number of advection cells for each edge
[in]advcellsforedgeInput: List of advection cells for each edge
[in]normalthicknessfluxInput: Thichness weighted velocitiy - (nVertLevels, nEdge)
[in]wInput: Vertical velocitiy - (nVertLevels+1, nEdges)
[in]layerthicknessInput: Thickness - (nVertLevels, nCells)
[in]verticalcellsizeInput: Distance between vertical interfaces of a cell - (nVertLevels, nCells)
[in]tend_layerthicknessInput: Tendency for thickness field - (nVertLevels, nCells)
[in]dtInput: Timestep
[in]meshpoolInput: Mesh information
[in,out]tendInput/Output: Tracer tendency
maxlevelcell_inInput - optional: Index to max level at cell center
maxleveledgetop_inInput - optional: Index to max level at edge with non-land cells on both sides
highorderadvectionmask_inInput - optional: Mask for high order advection
verticaldivergencefactor_inInput - optional: Denominator for vertical divergence. Given as an inverse which can be multiplied.
edgesignoncell_inInput - optional: Sign for flux from edge on each cell. Used for bit-reproducibility
