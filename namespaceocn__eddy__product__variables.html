Detailed Description

MPAS ocean analysis mode member: eddy_product_variables.

Author
Mark Petersen
Date
Oct 2016

MPAS ocean analysis mode member: eddy_product_variables computes: SSH^2 SSHSquared u^2 velocityZonalSquared v^2 velocityMeridionalSquared uT velocityZonalTimesTemperature vT velocityMeridionalTimesTemperature

In future, could additionally compute: uS velocityZonalTimesSalinity vS velocityMeridionalTimesSalinity uw velocityZonalTimesVertVel vw velocityMeridionalTimesVerVel

They you just stick these in the time-average AM. We already time average u,v, and all tracers. Then compute <v'T'> etc as a post processing step, same as POP, as follows:

<v'T'> = <vT> - <v><T> <(u')^2> = <u^2> - ^2
