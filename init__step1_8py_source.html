1#!/usr/bin/env python
2"""
3This script performs the first step of initializing the global ocean. This
4includes:
51. combining Natural Earth land coverage north of 60S with Antarctic
6 ice coverage or grounded ice coverage from Bedmap2.
72. combining transects defining cricial passages.*
83. combining points used to seed a flood fill of the global ocean.
94. create masks from land coverage.
105. create masks from transects.*
116. cull cells based on land coverage but with transects present
127. create flood-fill mask based on seeds
138. cull cells based on flood-fill mask
149. create masks from transects on the final culled mesh*
15* skipped if flag --with_critical_passages not present
16
17Optionally, the -p flag provides the path to the geometric_features
18repository, which is assumed to be the current directory by default.
19Also, the optional --with_cavities flag indicates that ice-shelf cavities
20are present and the grounded-ice mask from Bedmap2 should be used.
21The optional --with_critical_passages flag indicates that critical
22passages are to be opened. Otherwise, steps 2, 5 and 9 are skipped
