integer, intent(out)  ierr

Checks input fields and options for compatibility.

Author
Michael Duda
Date
22 October 2018

This routine checks the input fields and run-time options provided by the user for compatibility. For example, two run-time options may be mutually exclusive, or an option may require that a certain input field is provided.

A value of 0 is returned if there are no incompatibilities among the provided input fields and run-time options, and a non-zero value otherwise.
