18!.. This subroutine computes the moisture tendencies of water vapor,
19!.. cloud droplets, rain, cloud ice (pristine), snow, and graupel.
20!.. Prior to WRFv2.2 this code was based on Reisner et al (1998), but
21!.. few of those pieces remain. A complete description is now found in
22!.. Thompson, G., P. R. Field, R. M. Rasmussen, and W. D. Hall, 2008:
23!.. Explicit Forecasts of winter precipitation using an improved bulk
24!.. microphysics scheme. Part II: Implementation of a new snow
25!.. parameterization. Mon. Wea. Rev., 136, 5095-5115.
26!.. Prior to WRFv3.1, this code was single-moment rain prediction as
27!.. described in the reference above, but in v3.1 and higher, the
28!.. scheme is two-moment rain (predicted rain number concentration).
29!..
30!.. Beginning with WRFv3.6, this is also the "aerosol-aware" scheme as
31!.. described in Thompson, G. and T. Eidhammer, 2014: A study of
32!.. aerosol impacts on clouds and precipitation development in a large
33!.. winter cyclone. J. Atmos. Sci., 71, 3636-3658. Setting WRF
34!.. namelist option mp_physics=8 utilizes the older one-moment cloud
35!.. water with constant droplet concentration set as Nt_c (found below)
36!.. while mp_physics=28 uses double-moment cloud droplet number
37!.. concentration, which is not permitted to exceed Nt_c_max below.
38!..
39!.. Most importantly, users may wish to modify the prescribed number of
40!.. cloud droplets (Nt_c; see guidelines mentioned below). Otherwise,
41!.. users may alter the rain and graupel size distribution parameters
42!.. to use exponential (Marshal-Palmer) or generalized gamma shape.
43!.. The snow field assumes a combination of two gamma functions (from
44!.. Field et al. 2005) and would require significant modifications
45!.. throughout the entire code to alter its shape as well as accretion
46!.. rates. Users may also alter the constants used for density of rain,
47!.. graupel, ice, and snow, but the latter is not constant when using
48!.. Paul Field's snow distribution and moments methods. Other values
49!.. users can modify include the constants for mass and/or velocity
50!.. power law relations and assumed capacitances used in deposition/
51!.. sublimation/evaporation/melting.
52!.. Remaining values should probably be left alone.
53!..
54!..Author: Greg Thompson, NCAR-RAL, gthompsn@ucar.edu, 303-497-2805
55!..Last modified: 01 Aug 2016 Aerosol additions to v3.5.1 code 9/2013
56!.. Cloud fraction additions 11/2014 part of pre-v3.7

2860!..All other moments based on reference, 2nd moment. If bm_s.ne.2,

2866 + sa(4)*tc0*bm_s + sa(5)*tc0*tc0 &
2867 + sa(6)*bm_s*bm_s + sa(7)*tc0*tc0*bm_s &

2872 + sb(4)*tc0*bm_s + sb(5)*tc0*tc0 &

2900 + sb(5)*tc0*tc0 + sb(6)*cse(14)*cse(14) &
2901 + sb(7)*tc0*tc0*cse(14) + sb(8)*tc0*cse(14)*cse(14) &

2924 n0_min = min(n0_exp, n0_min)
2925 n0_exp = n0_min

2927 lamg = lam_exp * (cgg(3)*ogg2*ogg1)**obmg

2932 endif

2939 ilamr(k) = 1./lamr

2944!+---+-----------------------------------------------------------------+
2945!..Cloud water condensation and evaporation. Nucleate cloud droplets
2946!.. using explicit CCN aerosols with hygroscopicity like sulfates using
2947!.. parcel model lookup table results provided by T. Eidhammer. Evap
2948!.. drops using calculation of max drop size capable of evaporating in
2949!.. single timestep and explicit number of drops smaller than Dc_star
2950!.. from lookup table.
2951!+---+-----------------------------------------------------------------+
2952 do k = kts, kte
2953 orho = 1./rho(k)
2954 if ( (ssatw(k).gt. eps) .or. (ssatw(k).lt. -eps .and. &
2955 l_qc(k)) ) then
2956 clap = (qv(k)-qvs(k))/(1. + lvt2(k)*qvs(k))
2957 do n = 1, 3
2958 fcd = qvs(k)* exp(lvt2(k)*clap) - qv(k) + clap
2959 dfcd = qvs(k)*lvt2(k)* exp(lvt2(k)*clap) + 1.
2960 clap = clap - fcd/dfcd
2961 enddo
2962 xrc = rc(k) + clap*rho(k)
2963 xnc = 0.
2964 if (xrc.gt. r1) then
2965 prw_vcd(k) = clap*odt
2966!+---+-----------------------------------------------------------------+ ! DROPLET NUCLEATION
2967 if (clap .gt. eps) then
2968 if (is_aerosol_aware) then
2969 xnc = max(2., activ_ncloud(temp(k), w1d(k), nwfa(k)))
2970 else
2971 xnc = nt_c
2972 endif
2973 pnc_wcd(k) = 0.5*(xnc-nc(k) + abs(xnc-nc(k)))*odts*orho
2974
2975!+---+-----------------------------------------------------------------+ ! EVAPORATION
2976 elseif (clap .lt. -eps .AND. ssatw(k).lt.-1.e-6 .AND. is_aerosol_aware) then
2977 tempc = temp(k) - 273.15
2978 otemp = 1./temp(k)
2979 rvs = rho(k)*qvs(k)
2980 rvs_p = rvs*otemp*(lvap(k)*otemp*orv - 1.)
2981 rvs_pp = rvs * ( otemp*(lvap(k)*otemp*orv - 1.) &
2982 *otemp*(lvap(k)*otemp*orv - 1.) &
2983 + (-2.*lvap(k)*otemp*otemp*otemp*orv) &
2984 + otemp*otemp)
2985 gamsc = lvap(k)*diffu(k)/tcond(k) * rvs_p
2986 alphsc = 0.5*(gamsc/(1.+gamsc))*(gamsc/(1.+gamsc)) &
2987 * rvs_pp/rvs_p * rvs/rvs_p
2988 alphsc = max(1.e-9, alphsc)
