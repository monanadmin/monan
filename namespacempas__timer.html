Detailed Description

MPAS Timer module.

Author
Doug Jacobsen
Date
12/22/2015

This module provides developers with internal timer routines. Additionally it provides standard interfaces to additional timer libarries, such as tau or gptl. Timers are stored as trees of lists of timers.
