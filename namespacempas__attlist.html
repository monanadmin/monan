integer, intent(out), optional  ierr 
)

MPAS duplicate attribute list routine.

Author
Doug Jacobsen
Date
04/26/2016

This routine creates a copy of an attribute list, and returns it as destAttList. This routine assumes that destAttList is empty, however it will remove every attribute from destAttList before it attempts to copy into it.

Parameters
srcattlistInput: Source attribute list
destattlistOutput: Destination attribute list
[out]ierrOptional Output: Error code
