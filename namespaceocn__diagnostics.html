This module contains the routines for computing diagnostic variables, and other quantities such as vertAleTransportTop.

subroutine, public ocn_diagnostics::ocn_validate_state ( type (domain_type), intent(inout)  domain,
integer, intent(in), optional  timeLevel 
)

Ocean state validation routine.

Author
Doug Jacobsen
Date
08/11/2016

This routine validates that the ocean state is able to continue running with for the next time step. If a processor detects that it is unable to continue running, some diagnostic information is written out about it. This routine relies on the definition that a NaN does not equal itself for detecting issues with the state.
