Setup for lock exchange test case.

Author
Doug Jacobsen
Date
02/18/2014

This routine sets up the initial conditions for the lock exchange test case. It is setup in the y direction, such that everything in the southern half of the domain has a temperature of 5.0C and the northern half has a value of 30.0C. Salinity is setup as a constant 35PSU. No windstress is specified, and layerThickness is constant depending on the input parameter config_lock_exchange_bottom_depth.
