Detailed Description

Main driver for MPAS ocean analysis core in post-processing mode.

Author
Mark Petersen
Date
November 2013

This module contains the drivers for the MPAS ocean analysis core in post-processing mode. None of these routines are used in run-time mode by the ocean core.
