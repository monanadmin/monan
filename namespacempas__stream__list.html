integer, intent(out), optional  ierr 
)

Get streams from a stream list.

Author
Michael Duda, Doug Jacobsen
Date
08/06/2014

This function allows one to search through a stream list for a set of matching streams. In the first call to this function, the stream argument should be a null pointer. When called a single time, this function sets the stream argument to the first stream in the list that matches the streamPattern argument. Upon subsequent calls, searching begins at the first stream after the stream argument, to allow one to find all streams that match streamPattern. Each time a valid stream is found, this function returns .true. and if no more streams are found, it will return .false.
