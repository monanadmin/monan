Detailed Description

MPAS ocean tracer advection driver.

Author
Mark Petersen, David Lee, Doug Jacobsen
Date
October 2017

This module contains driver routine for tracer advection tendencys as well as the routines for setting up advection coefficients and initialization of the advection routines.
