Detailed Description

This module is responsible for the intial creation and setup of the block data structures.

Author
Doug Jacobsen
Date
05/31/12

This module provides routines for the creation of blocks, with both an arbitrary number of blocks per processor and an arbitrary number of halos for each block. The provided routines also setup the exchange lists for each block.

Phase 1 of block creation finalization.

Author
Doug Jacobsen
Date
05/31/12

This routine finalizes the block initialization processor. It calls mpas_block_allocate to allocate space for all fields in a block. Then the 0 halo indices for each element and the exchange lists are copied into the appropriate block. A halo update is required after this routine is called to make sure all data in a block is valid.

Fields and non-framework required dimensions are not setup after this routine returns. To complete setup, a call to mpas_block_creator_finalize_block_phase2 must follow.

Parameters
[in]nhalosInput: Number of halos for cell fields
blocklistInput/Output: Linked List of blocks
ncellssolveInput: nCellsSolve field information
nedgessolveInput: nEdgesSolve field information
nverticessolveInput: nVerticesSolve field information
[in]vertexdegreeInput: vertexDegree dimension
[in]maxedgesInput: maxEdges dimension
indextocellidInput: indexToCellID field information
indextoedgeidInput: indexToEdgeID field information
indextovertexidInput: indexToVertexID field information
