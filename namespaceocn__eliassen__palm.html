Detailed Description

MPAS ocean analysis core member: Eliassen-Palm Flux Tensor.

Author
Juan A. Saenz, Todd Ringler
Date
May 2015

This module contains the routines for computing the Eliassen and Palm Flux Tensor in buoyancy coordinates, and related quantities.

Compute Eliassen-Palm flux tensor.

Author
Juan A. Saenz, Todd Ringler
Date
May 2015

This routine conducts all computations required for the EPFT analysis member. Each time this AM is called, the instananeous ocean state is interpolated onto the target buoyancy values. The state is then accumulated into the ensemble average variable arrays (varEA). Based on the current estimate of the ensemble average, thickness-weight velocities are estimated along with the computation of the Eliassen-Palm flux tensor.

Parameters
[out]errOutput: error flag
