MPAS Pool Summary write routine.

Author
Doug Jacobsen
Date
07/29/2015

This routine writes out a summary of the contents of a pool that match a specific type. The memberType input argument can take the value of MPAS_POOL_FIELD, MPAS_POOL_CONFIG, or MPAS_POOL_PACKAGE. The recurseSubpools_in input argument defaults to false, but can be used to write a summary of subpool contents in addition to the current pool.

It's important to note that this routine iterates over a pool. So if it is called within a pool iteration loop, it could cause issues.
